# Devansh Lamaniya Portfolio Website

A premium, recruiter-focused personal portfolio website built using **HTML5, CSS3, and Vanilla JavaScript**.

This portfolio showcases my work as a **Python Developer**, **Data Analyst**, and aspiring **AI & Machine Learning Engineer**, highlighting my technical skills, projects, education, experience, and professional journey through a modern interactive user experience.

## Overview

This project was originally inspired by a website built using a different technology stack and was completely rebuilt from scratch using only:

* HTML5
* CSS3
* Vanilla JavaScript (ES6+)

The goal was to create a lightweight, high-performance portfolio without relying on frontend frameworks while maintaining a premium visual experience and smooth interactions.
